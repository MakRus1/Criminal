Требования для корректной работы: python.

На сайтах по нашей тематике не было API, поэтому мы прибегли к созданию парсера,
но для него и для корректной работы сайта нужен сервер, пришлось сделать его локальным. 

Для запуска сервера нужно просто выполнить файл RunServer.bat (При запуске автоматически запустится парсер)
Для запуска парсера нужно выполнить файл StartParser.bat 

Для остановки сервера или парсера нужно использовать сочетание клавиш ctrl+c
